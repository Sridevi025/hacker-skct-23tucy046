# Sandbox Environment Builder

## Roll Number
23tucy046

## Project Name
SKCT_23tucy046_SandboxBuilder

## Description
This project demonstrates a sandbox environment using Kali Linux and Metasploitable2. A port scanning tool using Nmap is implemented.

## Tools Used
- Kali Linux
- Metasploitable2
- Python
- Nmap

## How to Run

1. python3 setup_lab.py
2. python3 run_tool.py
3. python3 analyze_results.py

## Test Cases
- Metasploitable IP
- Localhost
- Invalid IP
